# Danne API

1. Clonar proyecto
2. ```yarn install```
3. Clonar el archivo ```.env.template```
4. Cambiar las variables de entorno
5. Levantar base de datos
```
docker-compose up -d

6. Ejecutar SEED
```
http://localhost:3000/api/seed
```
```
7. Comando para levantar proyecto
```
yarn start:dev 
```
